# PsychSync User Guide

## Getting Started

### 1. Creating an Account
1. Navigate to the registration page
2. Enter your email, full name, and password
3. Check your email for verification link
4. Click the verification link
5. Log in with your credentials

### 2. Creating Your First Team
1. Go to the Teams page
2. Click "Create Team"
3. Enter team name and description
4. Add team members by email
5. Assign roles (Owner, Admin, Member)

### 3. Creating an Assessment

#### From Scratch
1. Go to Assessments â†’ Create Assessment
2. Fill in basic information
3. Add sections
4. Add questions to each section
5. Choose question types:
   - Multiple Choice
   - Rating Scale
   - Text Response
   - Yes/No
   - Likert Scale
6. Save as draft
7. Preview and test
8. Publish when ready

#### From Template
1. Go to Templates
2. Browse or search for templates
3. Click "Use This Template"
4. Select team (optional)
5. Review and customize
6. Publish

### 4. Taking an Assessment
1. Navigate to assessment
2. Click "Take Assessment"
3. Read instructions
4. Answer questions
5. Navigate between sections
6. Save progress (auto-saves every 30 seconds)
7. Submit when complete

### 5. Viewing Results
After submission:
- View your score
- Read interpretation
- See subscale scores
- Print or download results
- Compare with previous attempts

### 6. Managing Team Assessments
Team admins can:
- Create team-specific assessments
- Assign assessments to members
- View team analytics
- Track completion rates
- Export team results

### 7. Analytics Dashboard
View performance metrics:
- Personal statistics
- Response history
- Score trends
- Time spent
- Completion rates

## Best Practices

### Creating Effective Assessments
1. **Clear Instructions**: Provide detailed instructions
2. **Logical Flow**: Organize questions into sections
3. **Appropriate Length**: 10-30 minutes recommended
4. **Question Variety**: Mix question types
5. **Required vs Optional**: Mark truly required questions only

### Team Management
1. **Role Assignment**: Give appropriate permissions
2. **Regular Reviews**: Check team analytics monthly
3. **Communication**: Discuss results with team
4. **Privacy**: Respect individual results

### Assessment Administration
1. **Pilot Testing**: Test with small group first
2. **Set Expectations**: Communicate time requirements
3. **Follow-up**: Provide feedback on results
4. **Confidentiality**: Maintain data privacy

## Troubleshooting

### Can't log in
- Check email verification
- Reset password if needed
- Clear browser cache
- Try different browser

### Assessment not saving
- Check internet connection
- Verify you're still logged in
- Try manual save button
- Contact support if persistent

### Results not displaying
- Ensure assessment was submitted
- Allow time for score calculation
- Refresh the page
- Check with assessment creator
